# Projeto SGE

## Manual do Software

O presente manual tem a intenção de explicar como funciona cada etapa e tela do programa, focando principalmente nas funções mais importantes para o gerenciamento de serviços de elétrica.

---

## Tela de Login
Na tela de login, existem dois campos: 
- **Usuário** (representado pela carta)
- **Senha** (representada pela chave)

Os campos são previamente adicionados no sistema, mas o administrador pode cadastrar mais usuários, se necessário.
